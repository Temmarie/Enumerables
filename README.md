# Enumerable Ruby methods

> The aim of this project was to rebuild some of the methods from the Ruby Enumerable Module and test if they function as the original.

## Methods rebuilt:

- .each
- .each_with_index
- .select
- .all?
- .any?
- .none?
- .count
- .map
- .inject
